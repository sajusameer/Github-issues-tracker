1️⃣ What is the difference between var, let, and const?

  Ans: In JavaScript var, let and const are all used to declare variables. However, they differ in scope, redeclaration, and reassignment.


  Var: var is the old JavaScript variable declaration keyword. It is function scoped, meaning that if you declare it inside a function, it can be used within that function.We can both re-declare and reassign variables with var. 

  Let: let was introduced in the ES6 version. It is block scoped, we can use it within  block. We can reassign variables with let, but we can't re-declare them.


  Const:  const is also introduced in ES6. It is also block scoped. A variable declared with const cannot be reassigned or re-declared, so it is usually used to hold a constant value.


  2️⃣ What is the spread operator (...)?

  Ans: In JavaScript, the Spread Operator (...) is used to expand the elements within an array or object. It is usually used to copy, merge, or append new data.
